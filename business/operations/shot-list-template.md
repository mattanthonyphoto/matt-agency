## Figure / Human Element (from scout notes)

> Shoot these AFTER core architectural documentation is complete. The 80/20 rule: 80% clean architecture, 20% human moments. These are your portfolio differentiators.

| # | Moment | Location | Figure | Lens | Notes | Done |
|---|--------|----------|--------|------|-------|------|
| 1 | | | | | | [ ] |

| 4 | | | | | | [ ] |
| 5 | | | | | | [ ] |

**On-set reminders:**
- Architecture is still the hero. The figure should occupy ~10-20% of the frame.
- Candid over posed. Direct movement ("walk toward the window"), not expressions ("look contemplative").
- One figure per composition. Multiple people dilute the impact.
- Shoot the same composition with and without the figure — give yourself options.
- No eye contact with camera. The person is experiencing the space, not performing for it.
