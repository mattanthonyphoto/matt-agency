# Post-Production & Delivery Workflow

## Import & Backup

1. Import all cards to Lightroom catalog
2. Backup raw files to external drive
3. Create project folder: `YYYY-MM — Client Name — Project Name`

## Culling

- First pass: reject obvious misses (focus, exposure, duplicates)
- Second pass: select hero shots + supporting images per room/space
- Target delivery count per package (see `pricing.md`)

## Editing

- Follow visual standards in `../team/style-guide.md`
- Natural color grading — true to materials
- Straighten all verticals
- Clean up construction debris, visible wires, staging imperfections
- Maintain consistency across the full set
- Twilight and drone shots color-matched to interior set

## Export

| Deliverable | Format | Resolution | Color Space |
|-------------|--------|------------|-------------|
| Web / portfolio | JPEG | 2400px long edge | sRGB |
| Print / publication | TIFF or JPEG | Full resolution | Adobe RGB |
| Social media | JPEG | 1080x1350 (4:5) | sRGB |
| Award submission | Per submission requirements | Full resolution | sRGB |

## Delivery

1. Upload to Dropbox client folder
2. Organize by: `Final / [Web]` and `Final / [Print]` subfolders
3. Send delivery notification email (see `../marketing/email-templates.md`)
4. Share Dropbox link with client

## Follow-Up Sequence

| Timing | Action |
|--------|--------|
| Day 0 | Gallery delivered — send notification |
| Day 3 | Check in — any questions or feedback? |
| Week 2 | Request testimonial / Google review |
| Month 1 | Ask about case study (if strong project) |
| Month 3 | Check in — new projects coming up? Retainer pitch if repeat client |

## Archive

- Raw files: external drive, organized by year > client > project
- Final edits: Dropbox (persistent client access)
- Remove working files from local machine after delivery confirmed
