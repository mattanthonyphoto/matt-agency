# Shoot Day Checklist

## Gear Check (Before Departure)

### Camera
- [ ] Primary body — charged, card formatted
- [ ] Backup body — charged, card formatted
- [ ] Wide angle lens (16-35mm or equivalent)
- [ ] Standard zoom (24-70mm)
- [ ] Tilt-shift lens (if architectural)
- [ ] Spare batteries (x3 minimum)
- [ ] Spare memory cards (x3 minimum)
- [ ] Tripod + L-bracket

### Drone
- [ ] Drone body — inspected
- [ ] Batteries charged (x3 minimum)
- [ ] Controller charged
- [ ] ND filters
- [ ] Transport Canada pilot certificate accessible
- [ ] Airspace checked (NRC Drone Site Selection Tool)
- [ ] NOTAM check completed

### Lighting & Accessories
- [ ] Speedlights / strobes (if interior)
- [ ] Light stands
- [ ] Color checker / gray card
- [ ] Lens cloths + blower
- [ ] Step stool (for elevated angles)
- [ ] Gaffer tape

### Logistics
- [ ] Address + parking confirmed
- [ ] Client contact number saved
- [ ] Shot list printed / on phone
- [ ] Weather checked (golden hour time noted)

## On-Site Protocol

### Arrival (30 min before shoot)
1. Walk the full space — assess natural light direction and quality
2. Identify hero compositions (exterior, key rooms, signature details)
3. Note any staging issues to flag with client
4. Plan shooting sequence (chase the light)

### Shot Framework by Project Type

**Completed Project:**
- Hero exterior (2-3 angles, include context/landscape)
- Key interior spaces (living, kitchen, primary suite, bathrooms)
- Architectural details (millwork, stairs, materiality)
- Environmental context (views, setting, streetscape)
- Twilight exterior (if scheduled)
- Drone aerials (if scheduled)

**Build & Team Content:**
- Progress documentation (wide + detail)
- Team at work (candid, not posed)
- Material close-ups (texture, craft)
- Site context (signage, equipment, environment)
- Safety gear on, professional presentation

**Award & Publication:**
- Every room, every angle — overshoot
- Straight verticals, zero distortion
- Clean sightlines — no personal items unless intentional
- Details that show design intent (joinery, hardware, lighting design)
- Include context shots that tell the project story

### Wrap-Up
- [ ] Review shots on-site — confirm nothing missed
- [ ] Back up cards to laptop (if available)
- [ ] Thank client, set delivery timeline expectation
- [ ] Note any follow-up items (reshoot needs, twilight scheduling)
