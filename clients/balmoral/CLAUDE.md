## SEO Strategy

### Phase 1: Foundation (Launch)
18 pages minimum:
- Homepage, About, Services hub, 4 service pages
- 10 individual project pages
- 3 location pages (Whistler, Pemberton, Squamish) for "custom home builder"
- 3 location pages for "home renovations"
- Contact, Reviews, Blog hub

Technical SEO at launch:
- HomeAndConstructionBusiness + Organization schema (JSON-LD)
- FAQPage schema on every service and location page
- AggregateRating schema with real reviews
- Unique title + meta description per page
- XML sitemap with image extensions
- llms.txt and ai.txt for AI search
- All images WebP, descriptive filenames/alt text
- Open Graph + Twitter Card per page
- Canonical URLs, robots.txt with sitemap ref

### Phase 2: Content Engine (Weeks 3-8)
2 blog posts/week targeting:
- Cost guides ("How much does it cost to build in Whistler/Pemberton?")
- Process content ("BC building permit process explained")
- Location comparisons ("Building in Pemberton vs Whistler")
- Project case studies (detailed write-ups of completed work)
- Building science ("BC Energy Step Code for homeowners")

### Phase 3: Authority (Ongoing)
- Collect Google reviews (target 15+ at 5 stars)
- Apply for Georgie Awards
- Client video testimonials
- Lead magnet: "Sea to Sky Home Building Guide" PDF
- Monthly newsletter

### Phase 4: Location Domination (Weeks 6-12)

## Architecture

- **Platform:** Squarespace — custom code injected per page into code blocks
- **Approach:** Same method used on mattanthonyphoto.com — full HTML/CSS per page block
- **Goal:** Lead generation machine. Every page earns traffic or converts visitors into inquiries.

## Design Language

The design must feel like a builder's brand, not a photographer's. Marc's logo is the primary reference point for all design decisions.

### Brand Assets
- **Header logo:** Balmoral-Header3.png (clean sans-serif wordmark, "BALMORAL CONSTRUCTION")
- **Footer mark:** Separate brand mark used in footer
- **Logo style:** Modern, minimal sans-serif. No icons or decorative elements. Professional and understated.

### Design Direction
- **Palette:** TBD — derive from logo. Current site is light/white with photography doing the heavy lifting. New palette should feel like premium construction — consider concrete, steel, warm wood, stone tones.
- **Typography:** TBD — match the logo's clean sans-serif energy. No ornate serifs. Strong, architectural type.
- **Layout:** Full-width sections, generous whitespace, alternating image/text blocks, photography-forward
- **Hero:** Cinematic full-viewport with project photography. Clear headline about what they build and where.
- **Navigation:** Minimal fixed header, clean dropdowns, scroll-aware
- **Interactions:** Subtle hover effects on portfolio cards, smooth scroll, clean transitions
- **Vibe:** Confident, clean, built to last. Not flashy. The craftsmanship speaks. Think high-end contractor, not luxury fashion.

### Method
- Same code-per-page-block approach used on mattanthonyphoto.com
- Full HTML/CSS injected into Squarespace code blocks
- Proven workflow, just different brand expression
