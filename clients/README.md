# Clients

Each client gets their own folder. Standard structure per client:

```
client-name/
  src/          # Source files, website content, assets
  workflows/    # Client-specific SOPs
  tools/        # Client-specific scripts
  deliverables/ # Final outputs before pushing to cloud
  .tmp/         # Scratch files, intermediate data
```

Adapt as needed per client — not every folder is required.
