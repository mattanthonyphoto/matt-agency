# Content Batching Workflow

SOP for turning one completed photo shoot into 15-20+ pieces of content across all platforms. This is the production engine behind the social media operation.

---

## Objective

Maximize the content output of every project by systematically working through the repurposing chain in a single batching session. One shoot should feed your social media for 4-8 weeks minimum.

---

## Required Inputs

- Delivered project gallery (web-sized images in Dropbox)
- Project metadata:
  - Client company name
  - Instagram handle(s) — builder, architect, designer
  - LinkedIn page(s) — builder, architect, designer
  - Project name
  - Location (city/area)
  - 2-3 standout design features (materials, site response, scale)
  - Any notable trades or suppliers
- BTS footage (phone camera roll from shoot day)
- Client testimonial (if received — can be added later)
- Screen recording of retouching session (if captured)

## Tools

- Image editor (Lightroom/Capture One for crops, Canva for graphics/PDFs)
- `hooks-scripts-library.md` — hook selection
- `caption-templates.md` — caption frameworks
- `hashtag-system.md` — pre-built hashtag groups
- Scheduling tool for publishing

## Expected Output

Per project, minimum:
- 3 Instagram feed posts (carousel, Reel, single/detail)
- 2 LinkedIn posts (document carousel, text insight)
- 8-15 Pinterest pins
- 1 Instagram Story series
- 1 journal post (2-4 weeks later)
- 2-4 bonus posts (testimonial, throwback, educational, before/after)

**Total: 15-25 content pieces from one shoot.**

---

## Step-by-Step Process

### Step 1: Prepare Assets (30 min)

**Select images for social use:**
- Pick 15-20 strongest images from the delivered gallery
- Include a mix: wide exteriors, interiors, detail shots, lifestyle/context
- If before/after exists (construction phase photos), pull those too

**Create platform crops:**

| Platform | Aspect Ratio | Dimensions | Use For |
|----------|-------------|------------|---------|
| Instagram Feed | 4:5 portrait | 1080 x 1350px | Carousels, singles |
| Instagram Reel | 9:16 portrait | 1080 x 1920px | Cover frames |
| Instagram Story | 9:16 portrait | 1080 x 1920px | Story slides |
| LinkedIn Image | 1.91:1 landscape | 1200 x 628px | Single image posts |
| LinkedIn Document | 4:5 portrait | 1080 x 1350px | PDF carousel slides |
| Pinterest | 2:3 portrait | 1000 x 1500px | Standard pins |
| Journal/Website | 16:9 landscape | 1600 x 900px | Blog header + inline |

**Organize files:**
```
[project-name]/social/
  ├── instagram/     (4:5 crops)
  ├── linkedin/      (PDF slides + 1.91:1 images)
  ├── pinterest/     (2:3 crops)
  ├── reels/         (raw BTS footage, retouching screencap)
  └── stories/       (casual BTS shots)
```

**Create LinkedIn document carousel PDF:**
- Open Canva (or equivalent) → Custom size 1080x1350px
- Slide 1: Hero image with hook text overlay (use brand fonts)
- Slides 2-6: One strong image per slide, minimal text overlay if any
- Slide 7: Takeaway text or subtle CTA
- Export as PDF

### Step 2: Write All Captions (60-90 min)

Work through each content piece systematically. Have these open side by side:
- `hooks-scripts-library.md` (pick hooks)
- `caption-templates.md` (fill templates)
- `hashtag-system.md` (grab pre-built sets)

**Piece 1 — Instagram Hero Carousel:**
1. Open hooks-scripts-library.md → Pillar 1: Project Showcases → Instagram Carousel Hooks
2. Pick the hook that best fits this project
3. Fill in the caption template from caption-templates.md → Project Reveal
4. Grab hashtag set from hashtag-system.md → Project Reveal Set A or B
5. Note which images go in which slide order (wide opener → interiors → details → hero repeat)

**Piece 2 — LinkedIn Document Carousel:**
1. Open hooks-scripts-library.md → Pillar 1 → LinkedIn Project Hooks
2. Pick a hook with a B2B angle (different from the Instagram hook)
3. Fill in caption-templates.md → LinkedIn Project Story
4. Grab LinkedIn hashtag set (3-5 only)
5. Write the "first comment" separately — this is where the portfolio link goes

**Piece 3 — Instagram BTS Reel:**
1. Review BTS footage — pick the most interesting 15-30 seconds
2. Open hooks-scripts-library.md → Pillar 2: Process & BTS → Instagram Reels/Stories Hooks
3. Plan the Reel: hook text (0-3 sec) → BTS footage (3-20 sec) → final image reveal (20-30 sec)
4. Write caption (1-2 sentences + hashtags)
5. Note audio choice if using trending audio

**Piece 4 — Pinterest Pins (batch all at once):**
1. For each of the 8-15 strongest images:
   - Write a keyword-rich title: "[Style] [Type] in [Location], BC | [Feature]"
   - Write description using caption-templates.md → Pinterest Project Pin
   - Assign to a board (see pinterest-playbook.md for board structure)
   - Set the destination URL to the specific project page on your website
   - Write alt text describing the image with keywords

**Piece 5 — Instagram Stories:**
1. Select 3-5 casual BTS shots or behind-the-scenes moments
2. Write brief text overlays (location tag, client tag, casual commentary)
3. Plan any interactive elements (poll: "Which room is your favourite?", question sticker)

**Piece 6 — Client Tag Post:**
1. Pick 1-2 hero images specifically designed for the client to reshare
2. Write caption that elevates the client's work (not yours)
3. Tag all collaborators in image and caption
4. This post should make the client proud enough to reshare to their audience

**Piece 7 — Educational Post:**
1. Identify one teaching moment from this project ("Why this angle works," "How natural light transforms this space," "The staging decision that made this shot")
2. Open hooks-scripts-library.md → Pillar 4: Educational
3. Write as carousel (IG) or text post (LinkedIn) using the relevant template
4. This can be scheduled 2-3 weeks after the reveal to keep the project relevant

**Piece 8 — Detail/Material Post:**
1. Pick 1 stunning detail shot (millwork, tile, hardware, light pattern)
2. Write a micro-caption (2-3 sentences max)
3. Tag the specific trade responsible (@millwork_shop, @tile_supplier)

**Piece 9 — Before/After (if available):**
1. Match construction-phase photos with final photography at the same angle
2. Create carousel (before slide → after slide) or Reel (beat-drop transition)
3. Write caption using Before/After template

**Piece 10 — Journal Post (write outline now, draft later):**
1. Outline the blog post: project overview, design story, 2-3 key images inline, link to service pages
2. Note target keywords for SEO
3. Schedule the full draft for 2-4 weeks after the initial social push
4. This journal post will generate 3-5 additional Pinterest pins when published

### Step 3: Load Into Scheduling Tool (30 min)

1. Open your scheduling tool (Buffer, Later, or content calendar spreadsheet)
2. Upload all images paired with their captions
3. Schedule according to the weekly posting calendar:

**Week 1 (the reveal week):**
- Monday: Instagram hero carousel
- Monday: LinkedIn document carousel
- Tuesday: Instagram BTS Reel
- Thursday: Instagram detail or educational post
- Daily: 3-5 Pinterest pins
- Shoot day or delivery day: Instagram Stories

**Week 2:**
- Monday: Client tag post (Instagram)
- Tuesday: Educational post (LinkedIn)
- Thursday: Before/after (Instagram)
- Daily: 3-5 more Pinterest pins (different images, different boards)

**Week 3-4:**
- Publish journal post → generate Pinterest pins from it
- Testimonial graphic when received
- Any remaining detail shots as single-image posts

**Month 3-6:**
- Queue throwback reshare with seasonal angle
- Case study post if project warrants it

4. Verify all tags, hashtags, and links are correct
5. Set all posts to their optimal times:
   - Instagram: 8-11am or 7-9pm Pacific
   - LinkedIn: Tuesday-Thursday, 8-10am Pacific
   - Pinterest: Evening and weekends (when people browse)

### Step 4: Quality Check (10 min)

Before finalizing the batch:

- [ ] Every Instagram carousel ends with the first image repeated
- [ ] Every Instagram post has 15-20 hashtags from the rotation sets
- [ ] Every LinkedIn post has NO links in the body (link in first comment)
- [ ] Every LinkedIn post has 3-5 hashtags max
- [ ] Every Pinterest pin has a keyword-rich title, 100-200 word description, and correct destination URL
- [ ] All client/collaborator tags are correct handles
- [ ] No two posts on the same platform have identical hooks
- [ ] Content is spread across 2-4 weeks, not clustered
- [ ] Mix of content pillars across the week (not all showcases in a row)
- [ ] Client has been notified / approved any posts that tag them (if required)

---

## Batching Multiple Projects

When you have 2+ projects to batch at the same time:

1. **Process each project individually** through Steps 1-2
2. **Interleave when scheduling** — alternate projects in the calendar so the feed has variety
3. **Stagger reveals** — don't post two hero carousels from different projects in the same week
4. **Use the weaker project** to fill educational and detail slots around the stronger project's reveal week

## Time Estimates

| Step | Duration | Frequency |
|------|----------|-----------|
| Asset preparation | 30 min | Per project |
| Caption writing (all pieces) | 60-90 min | Per project |
| Scheduling | 30 min | Per project |
| Quality check | 10 min | Per project |
| **Total per project** | **~2.5 hours** | Per project |
| Journal post draft | 60-90 min | 2-4 weeks after shoot |

**For a typical month with 2-3 projects:** 5-8 hours of batching produces 4-8 weeks of content across all platforms.
