Homepage mockups are standalone HTML files used as sales pitches — premium redesigns of prospect websites.

**Local storage:** `business/sales/<name>-homepage-mockup.html`
**Live deployment:** `matt-proposals` public repo on GitHub Pages
**URL pattern:** `https://mattanthonyphoto.github.io/matt-proposals/<name>/index.html`
**Repo:** `mattanthonyphoto/matt-proposals` (public, Pages enabled on main branch, root path)

**Existing mockups in matt-proposals:** balmoral, coppercreek, kombi, koze, lrd, sitelines, sonoma-design-build, summerhill, matt-anthony-photography, sacred-ways

**Design pattern:** Single-file HTML with embedded CSS/JS. Google Fonts, CSS variables, scroll-triggered reveals, mobile responsive. Placeholder gradients where real photos aren't available. Loader animation, scroll progress bar, sticky nav with scroll state change.

**Why:** These are used to cold-pitch builders/businesses by showing them what a modern redesign of their site could look like. Sent via email or DM as a live link.
